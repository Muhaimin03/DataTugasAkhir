Fungsi Menu Pada Peta

Layer Control

Untuk mengontrol dan mengelola lapisan data yang ditampilkan pada peta.

Zoom Control

Untuk mengubah tingkat pembesaran (zoom in) dan perkecilan (zoom out) pada peta.

Lokasi

Untuk mengetahui lokasi pengguna di permukaan tampilan peta.

Reset View

Memungkinkan pengguna untuk dengan cepat mengembalikan tampilan peta ke pengaturan awal atau tampilan default.

Informasi Melalui Popup

Lokasi Penting

Informasi tiap lokasi penting dapat diperoleh dengan mengklik pada tiap lokasi penting sehingga muncul popup. Nama lokasi penting dapat dilihat dengan mengarahkan kursor dan juga muncul otomatis pada skala zoom tertentu.

Jalur Pendakian

Informasi pada jalur pendakian dapat diperoleh dengan mengklik jalur pendakian sehingga muncul popup. Informasi pada jalur pendakian merupakan informasi jalur antara lokasi penting.

Waktu Tempuh

Untuk memperoleh informasi waktu tempuh, dapat mengaktifkan layer waktu tempuh pada control layers, kemudian informasi waktu tempuh akan diperoleh dengan mengklik tiap jalur pendakian.
